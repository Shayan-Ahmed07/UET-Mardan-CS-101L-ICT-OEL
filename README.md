# ICT-OEL
Tourism in Pakistan
Project Overview

Tourism in Pakistan is a static website developed using pure HTML that presents Pakistan’s natural landscapes, cultural heritage, historical sites, and religious landmarks.
The project was created as part of an academic open-ended lab assignment and focuses on applying fundamental HTML concepts while delivering informative and well-structured content.

The website uses text, images, and videos to provide an engaging overview of major tourist destinations across Pakistan without relying on CSS or JavaScript.

Objectives:

1.Demonstrate a strong understanding of basic HTML structure and semantic element
2.Highlight the importance of tourism to Pakistan’s economy and cultural identity
3.Present major tourist destinations in a clear, section-based layout
4.Integrate multimedia elements using standard HTML tags
5.Create an educational resource suitable for academic use

Project Structure:

The website is organized into the following main sections:

1. Importance of Tourism in Pakistan

Discusses the role of tourism in economic development
Highlights employment generation and infrastructure growth
Emphasizes cultural exchange and heritage preservation

2. Northern Areas of Pakistan

Overview of scenic regions in Gilgit-Baltistan and Khyber Pakhtunkhwa
Featured destinations include:

Hunza Valley
Skardu

Focus on natural beauty, adventure tourism, and mountain landscapes

3. Lahore – The Cultural Capital

Explores Lahore’s historical and cultural significance
Major attractions covered:

Badshahi Mosque
Shalimar Gardens

Emphasis on Mughal architecture and artistic heritage

4. Karachi – The City of Lights

Highlights Karachi as Pakistan’s largest city and economic hub
Key tourist sites:

Mausoleum of Quaid-e-Azam
National Museum of Karachi

Mix of modern urban life and historical importance

5. Historical Sites of Pakistan

Focuses on ancient civilizations and archaeological heritage
Featured locations:

Mohenjo-Daro
Taxila

Showcases early urban planning and Buddhist heritage

6. Religious Tourism in Pakistan

Covers religious diversity and spiritual touris
Important sites include:

Faisal Mosque
Data Darbar

Represents Islamic architecture and Sufi traditions

Technologies Used:

HTML5

HTML elements such as:

Headings (h1–h3)
Paragraphs and lists
Images (img)
Videos (video)
Horizontal rules (hr)
No external libraries, frameworks, CSS, or JavaScript were used.

How to Run the Project:
Download or clone the repository
Ensure the following folders exist in the same directory as the HTML file:

imagescomp/ (contains image files)
videos/ (contains video files)
Open the ICT OEL.html file in any modern web browser

Intended Use:

Academic assignments and lab submissions
Learning and practicing basic HTML
Informational content on Pakistan’s tourism
Demonstration of multimedia integration in HTML
